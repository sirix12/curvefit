## How to use Locally
1. Clone this repository.
2. Open `index.html` in your web browser. 
   - No server needed! It's pure HTML/JS.

## How to Deploy to GitHub Pages
1. Push this code to a GitHub repository.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, select **Source** as `Deploy from a branch`.
4. Select `main` (or `master`) as the branch and `/ (root)` as the folder.
5. Click **Save**.
6. Wait a moment, and GitHub will provide your live URL!
